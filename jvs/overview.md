# JVS — JAMMA Video Standard

Digital successor to the analog JAMMA edge connector. Uses a serial protocol (typically over RS-485) for I/O devices (controls, coin, switches, sensors, etc.).

**Key public references**:
- [sega.bsnk.me JVS documentation](https://sega.bsnk.me/ringedge/hardware/jvs/)
- [English translation of JVS 3rd edition (WIP)](http://daifukkat.su/files/jvs_wip.pdf)
- Shizmob/arcade-docs and various I/O board RE.

Many modern systems (Sega Naomi family and later, Namco System 246/256, Taito Type X variants, etc.) use JVS I/O boards that often still expose a JAMMA edge for cabinet compatibility.

Common Sega I/O part numbers: 838-xxxxx series.

See also conversion notes (JVS ↔ JAMMA adapters) in community forums and arcade-projects.

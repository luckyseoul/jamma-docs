# Sega Arcade Systems (JAMMA / JVS Generations)

Sega produced many of the most important JAMMA and later JVS boards.

### Classic JAMMA era
- System 16 / System 16B / System 16C
- System 18
- System 24
- System C / C2
- ST-V (Sega Titan Video — Saturn hardware in JAMMA form factor; cross-reference saturn-docs for shared architecture)

### 3D / High-end
- Model 1
- Model 2
- Model 3 (and Model 3 Step 1.5/2.0)

### JVS era
- Naomi / Naomi 2
- Chihiro
- Triforce (GameCube-based, with Namco/Nintendo)
- Lindbergh
- RingEdge / RingEdge 2
- ALLS
